# Codename

A Raycast extension that generates random codenames for your projects.

## Features

- Generate random codenames in the format `XX-11` (two uppercase letters + hyphen + two digits)
- Automatically copies the generated codename to your clipboard
- Shows a HUD notification with the copied codename

## Examples

- `NK-23`
- `LM-89`
- `OG-29`
- `QP-42`

## Usage

1. Open Raycast
2. Search for "Generate Codename"
3. Press Enter to execute
4. The codename is automatically copied to your clipboard

## Installation

### From Raycast Store

Search for "Codename" in the Raycast Store and click Install.

### Manual Installation

```bash
git clone https://github.com/vkboo/codename-raycast.git
cd codename-raycast
npm install
npm run dev
```
